# Razen

The AI employee that researches, writes, plans, and builds — the work of a full team, delivered in one chat.

Built with TanStack Start, Vite, Tailwind v4, Supabase Edge Functions, and Gemini 3 Flash via the Lovable AI Gateway.

## Modes

- **Research** — decomposes a question into parallel sub-questions, searches the web with citations, streams a live agent timeline, and synthesises a cited analyst memo.
- **Write** — editorial-grade studio with inline ghost-text autocomplete, focus mode, and version history.
- **Plan** — Outline · Kanban · Calendar with owners, priorities, due dates, drag-and-drop, and an AI Monday standup memo.
- **Build** — a real product studio: live sandboxed preview, multi-file projects, file tree, error console, plain-English iteration, version snapshots, fork, auto-fix-errors, and ZIP export.
